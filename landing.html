One popup for visual audits and asset capture.

The extension focuses on repeatable frontend research tasks that usually require several separate tools.

#

Design tokens

Extract colors, font families, typography styles, spacing, shadows, and border radius values from the active page.

T

Tech detection

Identify frameworks, CMS platforms, ecommerce systems, analytics scripts, CDNs, and UI libraries found in the page.

C

Color picker

Sample pixels with a magnifier, copy HEX and RGB values, and keep a short history of picked colors.

I

Image export

List page images, CSS backgrounds, and inline SVGs, then download files individually or bundled as a ZIP.

S

Screenshots

Capture the visible viewport, stitch full-page screenshots, or select a custom region with PNG, JPG, and WebP output.

{ }

CSS inspector

Hover elements to inspect computed styles, lock a target on click, and copy the full style block.

Built for fast teardown sessions.

Move from discovery to export without leaving the page you are studying.

Install as an unpacked extension.

This repository is already structured as a Chrome Manifest V3 extension. Load the folder directly during development.

Open Chrome extensions
  1. 1Open chrome://extensions in Chrome.
  2. 2Enable Developer mode in the top-right corner.
  3. 3Choose Load unpacked and select this project folder.
  4. 4Pin the extension and run it on the page you want to inspect.
